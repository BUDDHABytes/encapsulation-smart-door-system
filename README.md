# Encapsulation Smart Door System

## Introduction
This repository contains the solution for the Smart Door Access System assignment for the Object-Oriented Programming (OOP) course at UMaT.

## Task Summary
* The goal was to develop a smart door access system for ORIC staff that protects access codes from direct modification.
* Implemented a `Staff` class with:
    * A public attribute: `s_name`.
    * A private attribute: `__access_code`.
    * A `@property` decorator to manage access and updates.
* Validated the `access_code` length to ensure security.
